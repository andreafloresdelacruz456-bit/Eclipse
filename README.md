# 🚀 Desarrollo de Aplicaciones en Java | Portfolio

¡Bienvenido a mi repositorio principal de desarrollo! Aquí recojo mi evolución como programadora Java, enfocándome en escribir código limpio, escalable y bien estructurado.

En este repositorio encontrarás implementaciones que cubren los pilares de la programación profesional:

* **POO (Programación Orientada a Objetos):** Dominio de Abstracción, Encapsulamiento, **Herencia** y Polimorfismo para crear código reutilizable.
* **Patrones de Diseño:** Implementación del patrón **DAO (Data Access Object)** para la separación de la lógica de negocio y el acceso a datos.
* **Gestión de Datos:** Manejo de colecciones complejas, persistencia y estructuras de datos eficientes.
* **Entorno:** Desarrollo avanzado en **Eclipse IDE** con control de versiones integrado en **Git**.
Proyectos destacables:
* **Sistemas con Arquitectura DAO:** Implementación de persistencia y separación de responsabilidades.
* **Jerarquías de Herencia:** Modelado de objetos del mundo real optimizando la jerarquía de clases.
* **Interfaces y Abstract Classes:** Definición de contratos de software para sistemas desacoplados.
